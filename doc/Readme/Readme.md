## Memcached

### 基本信息



**WebSite** ： http://memcached.org/

**维基百科**： https://en.wikipedia.org/wiki/Memcached

**GitHub**： https://github.com/memcached/memcached



**免费和开源，高性能，分布式内存对象缓存系统**，本质上是通用的，但旨在通过减轻数据库负载来加速动态Web应用程序。
